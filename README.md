# 都道府県別の総人口推移グラフを表示するアプリ

## 環境構築

### node_module をインストール

```
npm i
```

or

```
yarn
```

### 環境変数を作成

```
vi .env
```

```.env
REACT_APP_ENDPOINT= URL を入れる
REACT_APP_API_KEY= apiKey を入れる
```

### ローカルサーバーを立てる

```
npm run start
```

or

```
yarn start
```

### build する

```
npm run build
```

or

```
yarn build
```

### test する

```
npm run test
```

or

```
yarn test
```
